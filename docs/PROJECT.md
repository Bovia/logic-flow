# LogicFlow 项目说明

> 个人投研决策档案 — 把「研究 → 决策 → 记录 → 复盘」串成一条可追溯的闭环。

## 定位

LogicFlow **不是**行情终端，也**不是**自动交易工具。它解决的是：投研逻辑写在哪里、平仓后怎么复盘、事件和决策如何关联、自己的逻辑胜率到底多少。

- **面向谁**：个人投研者、投顾研究员、需要记录决策过程的学习者
- **使用场景**：手机端随时查看持仓逻辑、补充归因事件、做结构化复盘
- **数据归属**：全部存在浏览器 `localStorage`，无需登录即可使用

## 核心能力

### 画布

- 活跃决策 / 历史决策分区展示
- 搜索过滤（名称、代码、推理文本）
- 创建决策卡：标的、推理、目标价 / 止损价、置信度、产业链关联
- 详情弹窗：平仓四态（持仓中 / 止盈 / 止损 / 逻辑失效）、置信度更新
- **实时行情**：拉取东方财富报价，展示现价与浮动盈亏

### 产业链

- 产业链列表与详情
- 上 / 中 / 下游节点管理
- 节点关联标的 Pills

### 归因

- 事件按月时间轴
- 执行 / 分析模式（分析模式展示逻辑验证说明）
- **点击展开**详情、关联决策卡跳转

### 档案

- 逻辑胜率 & 盈利胜率（双环仪表）
- 统计卡片：总交易、持仓数、平均置信度、逻辑失效次数
- **四步复盘模板**：选决策 → 拉取关联事件 → 逻辑验证结论 → 复盘备注
- 复盘记录列表

## 技术栈

| 层 | 选型 |
|----|------|
| 前端 | React 19 + TypeScript + Vite |
| 状态 | React Context + localStorage |
| 行情 | 东方财富 API（dev: Vite 代理 / prod: Vercel Serverless） |
| 部署 | Vercel 静态站点 |
| 作品集 | Portfolio Hub（`slug: logic-flow`） |

## 不在范围内

- 券商账户接入、自动下单
- 高频行情、Level-2 数据
- 多用户云同步（当前版本）
- 投资建议合规发布（工具仅作个人档案）
